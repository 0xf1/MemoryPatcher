# MemoryPatcher

Universal Process Memory Patcher for Windows systems.

Use space char to separate bytes. 

for ex:
> Srch. bytes:   **D0 1B 0E B0**
> 
> Repl. bytes:  **D0 00 0E B0**

Put TitanEngine.dll in the same directory as the main program.
